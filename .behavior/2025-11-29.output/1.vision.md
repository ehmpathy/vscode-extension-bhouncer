# output

## overview

bhouncer provides observability into its operations through vscode's output panel.

## capabilities

### output channel

- logs appear in vscode's output panel under "bhouncer"
- select "bhouncer" from the output dropdown to see live logs
- logs include timestamps, levels, and contextual data

## log format

```
[2024-01-15T10:30:00.000Z] [INFO] activating...
[2024-01-15T10:30:00.050Z] [INFO] enabled language server {"settingKey":"terraform.languageServer.enable"}
[2024-01-15T10:30:05.000Z] [DEBUG] pruning editors {"eligible":3,"closed":2}
```

## when output is enabled

- always enabled by default: output channel is created and receives all logs
- can be disabled via `bhouncer.output.enabled` setting
