# output

## domain objects

### OutputConfig

```typescript
interface OutputConfig {
  outputChannel: vscode.OutputChannel | null;
}
```

## domain operations

### createOutput

```typescript
createOutput(input: {
  enabled: boolean;
  createOutputChannel: (name: string) => vscode.OutputChannel;
}): Output
```

- if enabled:
  - creates output channel "bhouncer"
  - returns output that writes to both console and channel
- if not enabled:
  - returns output that writes to console only

## extension integration

### activation

```typescript
// check if output is enabled via settings (defaults to true)
const outputEnabled = vscode.workspace.getConfiguration('bhouncer').get('output.enabled', true);

// create output
const output = createOutput({
  enabled: outputEnabled,
  createOutputChannel: vscode.window.createOutputChannel,
});

// store in state for access
state.output = output;
```

### deactivation

```typescript
// dispose output channel
state.output?.dispose();
```
